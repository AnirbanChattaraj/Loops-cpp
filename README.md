# 📘 C++ Loops Practice Set (For + While)

This repository contains a well-structured set of **10 C++ problems focused on loops**, specifically using **`for` and `while` loops**. It is designed for who want to build strong fundamentals in iteration and problem-solving.

---

## 🎯 Objective

The goal of this practice set is to help you:

* Understand and use `for` and `while` loops effectively
* Learn when to use each type of loop
* Build logic using iteration
* Combine loops with basic conditions
* Solve number-based problems

---

## 📂 Problem List

### 🟢 Level 1: Basic Loop Usage

1. **Print 1 to N**
Print numbers from `1` to `n` using both `for` and `while` loops.

2. **Sum of First N Numbers**
Calculate the sum of first `n` natural numbers using both loops.

---

### 🟢 Level 2: Number Operations

3. **Count Digits**
Count the number of digits in a given number using a `while` loop.

4. **Reverse a Number**
Reverse a number using a `while` loop.

---

### 🟡 Level 3: Core Loop Logic

5. **Factorial**
Compute factorial of a number using:

* `for` loop
* `while` loop

6. **Multiplication Table**
Print the multiplication table of a number using both loops.

---

### 🟡 Level 4: Loop + Condition

7. **Even Numbers**
Print all even numbers from `1` to `n` using both loops.

8. **Sum of Even Numbers**
Calculate the sum of all even numbers between `1` and `n`.

---

### 🔵 Level 5: Slightly Advanced

9. **Palindrome Number**
Check whether a number is a palindrome.

10. **Sum of Digits**
Calculate the sum of digits of a number.

---


## 📌 Guidelines

* Try solving each problem using **both `for` and `while` loops** wherever applicable
* Focus on **logic building**, not memorization
* Keep your code clean and properly formatted
* Test your code with different inputs

---

## 🚀 Learning Outcome

By completing this set, you will:

* Gain confidence in using loops
* Understand iteration deeply
* Be ready to move into:

* Nested loops
* Pattern problems
* Functions
